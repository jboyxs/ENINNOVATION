
### 1. 高斯滤波 (Gaussian Blur)

**高斯滤波**适用于去除高频噪声，使图像更加平滑。通过调节滤波内核的大小和标准差，可以平衡去噪效果和图像细节保留。

```python
# 调整滤波内核大小和标准差，以改善图像平滑度
dpt_filtered = cv2.GaussianBlur(dpt_normalized, (5, 5), 2)  # 5x5的内核，标准差为2
```

- **内核大小**: `(5, 5)` — 可以尝试增大或减小，如 `(3, 3)` 或 `(7, 7)`。
- **标准差**: `2` — 增大标准差会导致更强的平滑效果，降低则会更保留细节。

### 2. 双边滤波 (Bilateral Filter)

**双边滤波**能保留边缘特征，同时平滑其他区域，适合处理具有边缘的物体。它有三个主要参数可以调整：

```python
# 调整双边滤波的参数，尝试不同的值以优化边缘保留效果
dpt_filtered = cv2.bilateralFilter(dpt_normalized, 9, 75, 75)
```

- **内核大小**: `9` — 控制滤波的邻域大小，较大的值能平滑更多的区域。
- **空间距离 (σ空间)**: `75` — 控制空间高斯函数的标准差，影响空间邻域范围。
- **颜色距离 (σ颜色)**: `75` — 控制强度差异的标准差，影响平滑强度。大的值会模糊更多细节，小的值则保留更多。

### 整合滤波功能后的代码

可以根据需要选择开启一个或两个滤波器，具体代码如下：

```python
# 1. 高斯滤波 (Gaussian Blur)
dpt_filtered = cv2.GaussianBlur(dpt_normalized, (5, 5), 2)  # 调整内核尺寸和标准差

# 2. 双边滤波 (Bilateral Filter)
dpt_filtered = cv2.bilateralFilter(dpt_normalized, 9, 75, 75)  # 内核为9，空间距离为75，颜色距离为75
```

### 3. 调整 Canny 边缘检测参数

调整 **Canny 边缘检测** 的阈值可以进一步控制边缘的检测灵敏度：

```python
# 调整Canny边缘检测的阈值，以检测更多或更少的边缘
edges = cv2.Canny(dpt_filtered, 50, 150)  # 提升低阈值为50，高阈值为150
```

- **低阈值**: `50` — 提高此值会减少噪声，但可能会遗漏一些边缘。
- **高阈值**: `150` — 降低此值将捕捉更多边缘，可能会带来噪声。

### 4. 调整形态学操作

形态学操作用于处理图像边缘，填充噪声带来的小空洞。可通过改变内核大小和操作次数来优化效果：

```python
# 调整形态学操作的内核大小和迭代次数
kernel = np.ones((5, 5), np.uint8)  # 增大内核为 5x5
edges_dilated = cv2.dilate(edges, kernel, iterations=2)  # 扩大迭代次数为2
```

- **内核大小**: `(5, 5)` — 增大内核可以填充更多的边缘区域。
- **迭代次数**: `2` — 增加迭代次数可以让边缘更加突出。

通过调节这些滤波和边缘检测参数，您可以减少误检并提高小球识别的精度。
